## 🤝 Como Contribuir?

Este é um projeto **100% open-source** e movido pela comunidade. Se você já fez algum teste técnico, lembra dos requisitos de algum processo seletivo ou achou algum repositório de empresa que ficou aberto, a sua contribuição é muito bem-vinda!

Para contribuir, siga os passos abaixo:

1. Faça um **Fork** deste repositório.
2. Crie uma branch para a sua contribuição: `git checkout -b meu-novo-desafio`.
3. Crie uma pasta dentro de `backend/` ou `frontend/` com o nome da empresa/desafio.
4. Adicione um arquivo `README.md` dentro dessa pasta detalhando os requisitos do desafio (se tiver o link do repositório original da empresa, inclua também).
5. Envie suas alterações: `git commit -m 'Adiciona desafio da Empresa X'`.
6. Faça o Push para a sua branch: `git push origin meu-novo-desafio`.
7. Abra um **Pull Request**.

## 💡 Como usar este repositório para treinar?

1. Escolha um dos desafios listados acima.
2. Tente resolvê-lo sem olhar soluções prontas. Foque em arquitetura, boas práticas, testes e organização de código.
3. Se quiser, você pode abrir uma *Issue* compartilhando a sua resolução para que outras pessoas da comunidade possam fazer um Code Review e te ajudar a evoluir!
